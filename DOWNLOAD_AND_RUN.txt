🚀 HOW TO RUN (3 EASY OPTIONS)
═══════════════════════════════

OPTION 1 - SIMPLEST (No Setup Required)
────────────────────────────────────────
1. Open this folder: aeterna-website
2. Double-click: index.html
3. Done! Website opens in your browser


OPTION 2 - PYTHON SERVER (Recommended)
───────────────────────────────────────
1. Open Terminal/Command Prompt
2. Type: cd path/to/aeterna-website
3. Type: python -m http.server 8000
4. Open browser: http://localhost:8000


OPTION 3 - NODE.JS
──────────────────
1. Open Terminal/Command Prompt  
2. Type: cd path/to/aeterna-website
3. Type: npx serve
4. Open the URL shown

🔧 CUSTOMIZATION

→ Add file to: attached_assets/
→ Reference in: script.js products
